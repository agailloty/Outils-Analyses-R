# Environnement Rstudio

## Ce réperoire est un exemple de ce à quoi ressemble votre environnement de travail le jour de l'examen. Vous pouvez d'ores et déjà vous familiariser avec cet environnement pour ne pas perdre du temps précieux le jour de l'examen. 
## Il est important de double-cliquer sur le fichier ayant l'extension `.Rproj` pour ouvrir le projet dans Rstudio. De cette façon vous n'aurez pas de problème pour lire le jeu de données.
